# Политика конфиденциальности

Дата вступления в силу: 08.04.2026

Приложение "Подписки" помогает пользователю отслеживать собственные подписки и расходы.

## Какие данные обрабатываются

- Названия подписок
- Стоимость подписок
- Даты списания
- Локальные технические данные, необходимые для работы уведомлений

## Как используются данные

- Для отображения списка подписок и расчёта ежемесячных и ежегодных расходов
- Для локальных напоминаний о предстоящих списаниях

## Где хранятся данные

- Все данные о подписках хранятся локально на устройстве пользователя в базе данных Room
- Приложение не отправляет эти данные на внешний сервер

## Уведомления

- Приложение может отправлять локальные уведомления о приближении даты списания
- Разрешение на уведомления используется только для этой функции

## Платежи и Premium

- Покупки Premium должны обрабатываться через Google Play Billing
- Платёжные данные пользователя приложением не собираются и не хранятся

## Передача данных третьим лицам

- Приложение не продаёт и не передаёт персональные данные третьим лицам

## Удаление данных

- Пользователь может удалить подписки внутри приложения
- При удалении приложения локальные данные могут быть удалены системой Android

## Контакты

- Email для вопросов по конфиденциальности: privacy@yourdomain.example

- Заменить контактный email на реальный
- Разместить этот текст по публичному HTTPS-URL
